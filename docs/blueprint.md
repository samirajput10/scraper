# **App Name**: Webmail Harvester

## Core Features:

- URL Input: Allows the user to input the URL of the website to scrape.
- Email Scraping: Scrapes email addresses from the specified website.
- Data Formatting Tool: Uses AI to ensure scraped emails are properly formatted and standardized; acts as a tool against erroneous data.
- Email Display: Displays the extracted email addresses in a list.
- Email Export: Allows the user to export the list of emails as a CSV file.

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5), suggesting clarity and precision, befitting a tool that extracts and organizes information.
- Background color: Very light blue (#E3F2FD), offering a clean and unobtrusive backdrop that doesn't distract from the content.
- Accent color: Royal blue (#1E88E5), used to highlight interactive elements such as buttons and links, enhancing usability.
- Font: 'Inter', a sans-serif font for its clean and readable style suitable for displaying data.
- Use simple, clear icons to represent actions like 'scrape,' 'export,' and 'format.'
- Clean and structured layout to easily visualize and manage extracted emails.
- Subtle animations for loading states and when exporting data.
